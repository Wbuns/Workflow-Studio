# Global Workspace Search

Workflow Studio v2.0.3 adds a project-wide search overlay opened with `Ctrl+K`.

The initial search index combines navigation destinations, Markdown documentation, project metadata, generated packages, AI snapshots, templates, and timeline activity. Results remain scoped to the active workspace and use existing secure open-path and navigation services.

The first release is read-only. It does not execute arbitrary files, commands, or content discovered through search.
